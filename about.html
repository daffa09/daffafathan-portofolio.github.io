About Me

Saya seorang programmer, dengan pengalaman lebih dari 2 tahun di dunia programming, dan dengan latar belakang programming.

Fokus utama saya terdapat pada developer aplikasi atau website beserta design nya, saya bersemangat untuk mengimplementasikan design web saya menjadi web asli ataupun design aplikasi menjadi aplikasi utuh untuk membantu mempermudah pekerjaan dunia industri.

Saya senang dalam membuat suatu aplikasi yang berguna untuk membatu pengguna (user) dalam bekerja di dunia industri dan menemukan pengalaman baru bagi saya untuk bekerja sama dengan orang lain maupun bekerja sendiri dalam membuat produk digital.

Mempelajari hal yang baru merupakan hal favorit saya, terutama dalam perkembangan teknologi. Sehingga saya dapat mempelajari hal baru sekaligus mencari peluang untuk membuat sesuatu yang bermanfaat bagi masyarakat, kolaborasi dengan team juga menjadi hal yang saya sukai karena dapat bertukar pendapat untuk membuat aplikasi yang lebih efektif.

Jika tidak ada projects dengan client, saya menikmati waktu dengan membuat project sendiri untuk melatih kemampuan saya.
